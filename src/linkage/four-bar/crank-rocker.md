# 曲柄搖桿機構<br />(Crank-Rocker Mechanism)

![曲柄搖桿動畫](/images/linkage/曲柄搖桿動畫.gif)

## 構成條件

- 曲柄 `AB` 為最短桿
- 最短桿加上對偶桿（相鄰桿）之一小於另外兩桿之和

::: tip
AB + BC < CD + AD

AB + AD < BC + CD
:::
